# STX Utils

## Math
- `pow`: Exponentiation
- `sqrt`: Integer square root
- `abs-diff`: Absolute difference

## List
- `contains`: Check if list contains element
- `sum-list`: Sum all elements
- `filter-zeros`: Remove zeros

## Usage
```clarity
(contract-call? .stx-utils pow u2 u10) ;; 1024
(contract-call? .stx-utils sqrt u144)  ;; 12
```
